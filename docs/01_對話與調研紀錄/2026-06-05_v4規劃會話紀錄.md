## Decisions made (canonical list)

### Stack
- **M1 inference**: `ollama` (Metal) + `unsloth/gemma-4-e2b-it-GGUF` Q4_K_XL
- **Workstation inference**: `llama.cpp` (Vulkan) + `Ex0bit/Qwen3.6-27B-PRISM-PRO-DQ` Q4
- **Workstation backup model**: `mudler/Carnice-Qwen3.6-MoE-35B-A3B-APEX-MTP-GGUF` I-Compact 17GB
- **Agent**: `NousResearch/hermes-agent` (MIT, 162K★, MCP-native)
- **Memory**: `AxDSan/mnemosyne` (SQLite-only, BEAM, 98.9% LongMem)
- **Vector store**: `Qdrant` single-node, named vectors `text_vec(768) + image_vec(512)`
- **Skill spec**: `agentskills.io` (Anthropic + Hermes native)
- **Tool protocol**: MCP (filesystem, Qdrant, tags, shell)
- **Frontend**: `Open WebUI` + `Hermes CLI` dual interface
- **Network**: `Tailscale` mesh, 127.0.0.1 binding
- **Vision (M1)**: `MobileCLIP S0` ONNX + CoreML
- **Embeddings (M1)**: `nomic-embed-text`
- **ASR (M1)**: `whisper.cpp tiny`
- **ASR (workstation)**: `faster-whisper large-v3` on-demand
- **Video slicing**: `PySceneDetect 0.7` AdaptiveDetector + 10s cap

### Operations
- Service management: `launchd` (macOS) + `systemd` (Linux)
- Single worker indexer (v3 invariant)
- Phase 5 (optional): `llama.cpp RPC`, `KTransformers`, `Graphiti`

### RAM budgets
- **M1 peak** (8GB): macOS 2.5G + Hermes 500MB + Ollama/Gemma 2.5G + nomic 0.3G + Qdrant 50MB + Indexer 500MB on-demand = **~7.5GB**
- **Workstation** (48GB): Qwen3.6-27B-PRISM ~13.7GB in A770 16GB VRAM + 2-3GB KV + 4GB system + **25GB free**
