# 評估測試集 v2 — 100 Ground-Truth Queries

> **Purpose**: A reproducible, ground-truth evaluation set for the Personal AI Second Brain.
> **Size**: 100 queries across 8 categories
> **Format**: Each entry has `id`, `query`, `category`, `expected_docs` (list of file_ids), `difficulty`

---

## Annotation format

```json
{
  "id": "T001",
  "query": "Find photos of my dog at the beach",
  "category": "cross-modal",
  "expected_docs": ["2024-07-15_dog-beach.jpg", "2024-08-03_dog-beach-2.jpg"],
  "difficulty": "medium",
  "expected_answer_contains": ["dog", "beach"],
  "notes": "Both photos have a golden retriever; 'beach' is in EXIF tags"
}
```

The full JSONL is in [`../../tests/eval_v2_100.jsonl`](../../tests/eval_v2_100.jsonl).
This file lists 100 entries in human-readable form for annotation review.
